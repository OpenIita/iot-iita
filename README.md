# iotkit-parent

#### 介绍
iot系统后台项目，包含了mqtt认证服务、mqtt接入服务、web管理后台、规则引擎、第三方接入服务等模块。


#### 软件架构
软件架构说明
本系统采用springboot、mongodb、mqtt（EMQX）、keycloak等框架和第三方软件
